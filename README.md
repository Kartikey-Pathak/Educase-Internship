# 📱 Educase Assignment - ReactJS Frontend UI

This is the submission for the **Frontend Internship Assessment**. The task was to replicate a given Adobe XD mobile design using **React JS**, ensure pixel-perfect UI, seamless navigation, and deploy the final result on a hosting platform like Vercel or Netlify.
